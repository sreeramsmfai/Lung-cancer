About This Project

This project uses a Vision Transformer (ViT) model to classify lung X-ray images into:

  • Adenocarcinoma
  • Large Cell Carcinoma
  • Squamous Cell Carcinoma
  • Normal (Healthy Lungs)

This project is titled "Deep Learning Model For Early Lung Cancer Detection Via CT Imaging And Vision Transformer". The system is developed using HTML, CSS, Bootstrap, and JavaScript for the frontend, with a Python backend powered by the Flask framework. The goal is to propose an automated system for the early detection of lung cancer by leveraging Deep Learning, Image Processing, and the Vision Transformer (ViT) algorithm. A dataset of medical images, including various lung cancer types (adenocarcinoma, large cell carcinoma, squamous cell carcinoma) and healthy lung tissue, is used for classification. Feature extraction is applied to identify essential characteristics of each image. The ViT model, which excels at capturing spatial relationships in images, optimizes the classification accuracy. Several machine learning models are trained and evaluated with performance metrics such as precision, recall, and F1-score. The best-performing model is selected for deployment. A user-friendly interface is developed for healthcare professionals to diagnose lung cancer in real-time, enhancing diagnostic efficiency and accuracy in clinical settings.
